# VSCode Extension Project

A Visual Studio Code extension.

## Setup

```bash
npm install
```

## Development

```bash
npm run compile
```

## Testing

Press F5 in VSCode to open a new Extension Development Host window.

## Building

```bash
npm run package
```

## Project Structure

- `.claude/` - Claude AI configuration and notes
- `.data/` - Sample data files
- `src/` - Source code
- `out/` - Compiled JavaScript output
